## Set up for development
### Prerequisites
- Go version at least 1.13

### Set up
1. Enable Go module in go env
```
export GO111MODULE=on
```

2. Download dependencies
```
go mod download
```

3. Run code
```
go run .
```
or 
```
go run main.go
```
